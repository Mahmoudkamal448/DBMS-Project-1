# DBMS Project

DBMS is a Bash script DataBase Management System

# installation & Run
Use the version control tool [git](https://git-scm.com/) to clone the project.

    git clone git@github.com:AbdEl-RahmanKhaled/DBMS-Project.git
    cd DBMS-Project
    ./run.sh

Or press "Code" button then chose "Download zip"

# DataBase layer

- Create Database
- List DataBase
- Connect To DataBase
- Drop DataBase

# Table Layer

- Crate Table
- List Tables
- Drop Table
- Select Table

# Record Layer

- Insert Into Table
- Select From Table
- Delete From Table
- Update Table
